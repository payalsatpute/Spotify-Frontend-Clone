# Spotify Frontend UI Clone 🎧

This project is a **Spotify Frontend UI Clone** built using **HTML and CSS**.
It replicates the visual design of Spotify's interface, including the layout, navigation, and music player section.

⚠️ This project is **only a frontend design** and does **not include actual music playback functionality**.

---

## Features

* Spotify-like user interface
* Sidebar navigation menu
* Music player layout
* Album section
* Responsive design structure
* Clean and modern UI

---

## How to Run the Project

1. Clone the repository

```
git clone https://github.com/your-username/spotify-frontend-clone.git
```

2. Open the project folder

3. Open **Updated_Clone.html** in your browser.
